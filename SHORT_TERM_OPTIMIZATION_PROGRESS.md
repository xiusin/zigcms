## 📈 性能优化

### 1. 组件懒加载
所有弹窗和抽屉组件都使用 `v-if` 或 `v-model:visible` 控制，只在需要时渲染。

### 2. 数据计算优化
统计数据使用 `computed` 计算，自动缓存结果，避免重复计算。

### 3. 批量操作优化
批量操作使用单个 API 请求，而不是多个请求，减少网络开销。

### 4. JSON 格式化优化
JSON 格式化使用 `try-catch` 包裹，避免格式化失败导致页面崩溃。
