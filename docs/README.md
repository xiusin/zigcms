## 🚀 使用方法

### 本地预览

1. 直接在浏览器中打开：
   ```bash
   open docs/index.html
   ```

2. 或使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   cd docs
   python3 -m http.server 8000
   # 访问 http://localhost:8000
   
   # 使用 Node.js
   npx http-server docs -p 8000
   ```

### 部署到服务器

将 `docs` 目录上传到您的 Web 服务器即可。

## 🔧 自定义

### 修改配色

在 `<style>` 标签的 `:root` 部分修改 CSS 变量：

```css
:root {
    --primary: #0078d4;           /* 主色 */
    --primary-hover: #106ebe;      /* 主色悬停 */
    --primary-light: #e6f3ff;      /* 主色浅色 */
    --success: #107c10;            /* 成功色 */
    --warning: #ff8c00;            /* 警告色 */
    --danger: #e81123;             /* 危险色 */
    /* ... 更多颜色变量 */
}
```

### 添加新章节

1. 在侧边栏添加链接：
```html
<li><a href="#new-section">新章节</a></li>
```

2. 在主内容区添加章节：
```html
<section id="new-section" class="section">
    <h2>新章节标题</h2>
    <p>内容...</p>
</section>
```
