## 功能特性

### 1. CSRF 防护

**核心功能**:
- ✅ Token 生成和验证
- ✅ Cookie 设置（HttpOnly + SameSite）
- ✅ 安全方法白名单（GET、HEAD、OPTIONS）
- ✅ 路径白名单配置
- ✅ 会话关联验证

**配置参数**:
- Token 长度: 32 字节
- Token 有效期: 1 小时
- Cookie 名称: csrf_token
- 请求头名称: X-CSRF-Token

### 2. 速率限制

**核心功能**:
- ✅ 全局限流（1000 次/分钟）
- ✅ IP 限流（100 次/分钟）
- ✅ 用户限流（200 次/分钟）
- ✅ 端点限流（可配置）
- ✅ 白名单/黑名单支持
- ✅ 统计信息查询

**算法**: 滑动窗口

**端点配置**:
- 登录接口: 5 次/分钟
- AI 生成: 10 次/分钟

### 3. 权限控制（RBAC）

**核心功能**:
- ✅ 基于角色的访问控制
- ✅ 权限检查（单个/任一/所有）
- ✅ 超级管理员支持
- ✅ 公开路径配置
- ✅ 用户权限上下文

**权限定义**:
- 测试用例: 查看、创建、更新、删除、执行、批量操作
- 项目: 查看、创建、更新、删除、归档
- 模块: 查看、创建、更新、删除、移动
- 需求: 查看、创建、更新、删除、关联
- 反馈: 查看、创建、更新、删除、分配、跟进
- 统计: 查看、导出
- AI: 生成

### 4. 安全监控

**核心功能**:
- ✅ 安全事件记录
- ✅ 异常访问检测
- ✅ 自动告警（10 次/分钟）
- ✅ 自动封禁（20 次/分钟）
- ✅ 统计分析

**事件类型**:
- 登录失败/成功
- 权限拒绝
- 速率限制触发
- SQL 注入尝试
- XSS 攻击尝试
- CSRF 攻击尝试
- 敏感操作
- 异常访问
- 数据泄露风险

**严重程度**:
- 低（Low）
- 中（Medium）
- 高（High）
- 严重（Critical）

### 5. 审计日志

**核心功能**:
- ✅ 操作记录
- ✅ 数据变更记录（操作前/操作后）
- ✅ 失败操作记录
- ✅ 多维度查询
- ✅ 导出功能

**记录内容**:
- 用户信息（ID、用户名）
- 操作信息（类型、描述）
- 资源信息（类型、ID、名称）
- 数据变更（操作前、操作后）
- 请求信息（IP、User-Agent）
- 结果信息（成功/失败、错误信息）
- 时间戳

## 技术架构

### 系统架构图

```
┌─────────────────────────────────────────────────────────┐
│                    前端层 (Vue 3)                        │
│  ├─ 安全监控仪表板 (Dashboard)                          │
│  ├─ 审计日志查询 (Audit Log)                            │
│  └─ 告警管理 (Alerts)                                   │
└─────────────────────────────────────────────────────────┘
                          ↓ HTTP API (23 个路由)
┌─────────────────────────────────────────────────────────┐
│                  API 层 (Controllers)                    │
│  ├─ SecurityEvent Controller (6 个端点)                 │
│  ├─ AuditLog Controller (5 个端点)                      │
│  └─ Alert Controller (12 个端点)                        │
└─────────────────────────────────────────────────────────┘
                          ↓ 中间件处理
┌─────────────────────────────────────────────────────────┐
│                中间件层 (Middleware)                     │
│  ├─ CSRF Protection (Token 验证)                        │
│  ├─ Rate Limiter (速率限制)                             │
│  └─ RBAC (权限检查)                                     │
└─────────────────────────────────────────────────────────┘
                          ↓ 服务调用
┌─────────────────────────────────────────────────────────┐
│              基础设施层 (Infrastructure)                 │
│  ├─ Security Monitor (安全监控)                         │
│  └─ Audit Log Service (审计日志)                        │
└─────────────────────────────────────────────────────────┘
                          ↓ 数据持久化
┌─────────────────────────────────────────────────────────┐
│                  数据层 (Database)                       │
│  ├─ security_events (安全事件表)                        │
│  ├─ audit_logs (审计日志表)                             │
│  ├─ ip_bans (IP 封禁表)                                 │
│  ├─ alert_rules (告警规则表)                            │
│  └─ alert_history (告警历史表)                          │
└─────────────────────────────────────────────────────────┘
```

### 数据流图

```
用户请求
   ↓
速率限制检查 → 超限 → 拒绝 + 记录事件
   ↓ 通过
CSRF 验证 → 失败 → 拒绝 + 记录事件
   ↓ 通过
权限检查 → 无权限 → 拒绝 + 记录事件
   ↓ 通过
业务逻辑处理
   ↓
审计日志记录
   ↓
安全监控分析
   ↓
响应返回
```
