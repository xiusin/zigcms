## 🚀 性能优化

### 1. 组件懒加载
- 所有弹窗和抽屉组件使用 `v-if` 或 `v-model:visible` 控制
- 只在需要时渲染，减少初始加载时间

### 2. 数据计算优化
- 统计数据使用 `computed` 计算，自动缓存结果
- 避免重复计算，提升响应速度

### 3. 批量操作优化
- 批量操作使用单个 API 请求
- 减少网络开销，提升处理速度

### 4. JSON 格式化优化
- JSON 格式化使用 `try-catch` 包裹
- 避免格式化失败导致页面崩溃

### 5. 移动端优化
- 使用卡片式布局替代表格（移动端）
- 隐藏非关键列，减少渲染负担
- 优化触摸交互，提升用户体验

## 📚 使用指南

### 1. 查看告警详情

**操作步骤**:
1. 在告警列表中点击"详情"按钮
2. 或在通知面板中点击告警项
3. 查看完整的告警信息
4. 查看关联的安全事件
5. 查看处理记录（如果已处理）

**可用操作**:
- 导出告警详情为 JSON
- 复制告警ID
- 处理告警

### 2. 处理单个告警

**操作步骤**:
1. 在告警列表中点击"处理"按钮
2. 或在详情抽屉中点击"处理告警"按钮
3. 选择处理结果（标记已处理、忽略、升级）
4. 输入处理说明（必填）
5. 如果选择升级，选择升级对象
6. 可选：上传附件
7. 点击确定提交

**处理方式**:
- **标记已处理**: 告警已解决，记录处理说明
- **忽略**: 告警为误报或不需要处理
- **升级**: 告警需要更高级别的团队处理

### 3. 批量操作告警

**操作步骤**:
1. 在告警列表中勾选多个告警
2. 批量操作栏会自动显示
3. 选择操作类型

**支持的操作**:
- **批量处理**: 将选中的告警标记为已处理
- **批量忽略**: 将选中的告警标记为已忽略
- **批量导出**: 导出选中的告警为 JSON
- **批量删除**: 删除选中的告警（需确认）

### 4. 使用键盘快捷键

**常用快捷键**:
- `Ctrl + F`: 快速搜索
- `Ctrl + R`: 刷新列表
- `Ctrl + A`: 全选告警
- `Escape`: 取消选择
- `Delete`: 删除选中项
- `Alt + ←/→`: 翻页

**查看帮助**:
- 按 `Shift + ?` 显示快捷键帮助
- 或点击页面右上角的"快捷键"按钮

### 5. 移动端使用

**特性**:
- 自动适配屏幕尺寸
- 卡片式布局（更适合触摸）
- 优化的触摸交互
- 隐藏非关键信息

**操作**:
- 点击卡片查看详情
- 滑动查看更多告警
- 使用筛选快速定位
