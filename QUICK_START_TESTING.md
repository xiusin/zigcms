## ❌ 如果测试失败

### 常见问题 1：服务未启动

**错误**：
```
✗ 无法连接到服务
```

**解决**：
```bash
# 确认服务已启动
zig build run
```

### 常见问题 2：数据库未初始化

**错误**：
```
✗ 数据库连接失败
```

**解决**：
```bash
# 执行数据库迁移
sqlite3 data/zigcms.db < migrations/20260302_create_quality_center.sql
```

### 常见问题 3：路由未注册

**错误**：
```
✗ 路由未注册或配置错误
```

**解决**：
- 检查 `src/api/bootstrap.zig` 中的路由注册
- 确认控制器已正确实现
- 重新编译并启动服务
